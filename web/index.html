WRITE-UP PICOCTF: DISKO 1

Category: Forensic

Difficulty: Easy

Description: Can you find the flag in this disk image? Download the disk image here.

Hint: Maybe Strings could help? If only there was a way to do that?.

Solution

Pada tantangan kali ini, kami diberikan sebuah file disk image. Kami mengunduh file tersebut dan diketahui masih terkompresi dengan format .gzip. Oleh karena itu, kami mengekstraknya menggunakan perintah gzip.

Setelah proses ekstraksi selesai, diperoleh sebuah file disk image dengan format .dd. Sesuai dengan hint yang diberikan, kami menggunakan perintah strings pada file tersebut, kemudian dikombinasikan dengan grep pico menjadi strings disko-1.dd | grep pico untuk memfilter teks yang sesuai dengan format flag.

Boommm, flag ditemukan.
